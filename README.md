# Telegram-бот регистрации участников «Бархатный путь»

Готовый Telegram-бот на **aiogram 3** для регистрации участников мероприятия.

## Что умеет

- регистрация через кнопку **Старт**
- запрос имени
- запрос согласия на обработку персональных данных
- запрос номера телефона через кнопку Telegram
- автоматическое присвоение **уникального порядкового номера**
- запрет на повторную регистрацию одного и того же пользователя
- запрет на повторное использование одного и того же номера телефона
- меню участника:
  - **Моя информация**
  - **Меню** — отправка картинки с меню вечера
  - **Запись на личную консультацию**
  - **Назад**
- админ меню:
  - выгрузка участников в **Excel (.xlsx)**
  - выбор периода выгрузки: **сегодня / неделя / месяц**
  - **сброс базы** по паролю с подтверждением

## Быстрый запуск без Docker

### 1. Клонируйте репозиторий

```bash
git clone <YOUR_REPO_URL>
cd barhat_bot
```

### 2. Создайте виртуальное окружение

```bash
python -m venv venv
source venv/bin/activate
```

Для Windows:

```bash
venv\Scripts\activate
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Создайте `.env`

Скопируйте шаблон:

```bash
cp .env.example .env
```

Заполните переменные:

```env
BOT_TOKEN=your_telegram_bot_token
ADMIN_IDS=123456789
RESET_PASSWORD=your_strong_password
CONSULTATION_PHONE=+7 (999) 123-45-67
CONSULTATION_TEXT=Для записи на личную консультацию свяжитесь с организатором по телефону:
TIMEZONE=Europe/Berlin
MENU_IMAGE_PATH=assets/menu_today.jpg
DB_PATH=participants.db
EXPORTS_DIR=exports
```

## Как узнать свой Telegram user ID

Добавьте любого бота, который показывает ID, или используйте временный хендлер/бот для логирования `message.from_user.id`.

## 5. Положите картинку меню

Файл должен лежать здесь:

```text
assets/menu_today.jpg
```

## 6. Запуск

```bash
python run.py
```

## Как открыть админ меню

Администратору доступны:

- команда `/admin`
- кнопка **Админ меню** в обычном меню пользователя

Доступ получают только ID из `ADMIN_IDS`.

---

## Excel-выгрузка

Файлы выгрузки сохраняются в папку:

```text
exports/
```

Выгрузка содержит:

- внутренний ID
- уникальный номер участника
- имя
- телефон
- Telegram user ID
- username
- согласие на ПД
- дату регистрации

---

## Сброс базы

Сброс доступен только из админ меню.

Логика:

1. нажать **Сброс базы**
2. ввести пароль `RESET_PASSWORD`
3. подтвердить сброс кнопкой **ПОДТВЕРЖДАЮ СБРОС**

После очистки нумерация участников снова начинается с `1`.

---

## Важно

- база данных — **SQLite** (`participants.db`)
- состояния диалога хранятся в памяти процесса, поэтому при перезапуске бота текущий незавершенный диалог сбросится
- уже зарегистрированные участники при этом **не теряются**, потому что лежат в SQLite
