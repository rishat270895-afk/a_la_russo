# Telegram-бот регистрации участников

Простой проект без `.env`. Все настройки лежат прямо в `bot.py`.

## Что умеет бот
- регистрация участника в Telegram
- запрос имени
- согласие на обработку персональных данных
- запрос номера телефона
- присвоение уникального порядкового номера
- защита от повторной регистрации по Telegram ID
- защита от дублирования по номеру телефона
- меню участника:
  - Моя информация
  - Меню вечера
  - Запись на личную консультацию
  - Назад
- админ-меню:
  - выгрузка в Excel за сегодня
  - выгрузка в Excel за неделю
  - выгрузка в Excel за месяц
  - ресет базы по паролю

## Структура
- `bot.py` — весь код бота
- `requirements.txt` — зависимости
- `assets/menu.jpg` — картинка меню вечера
- `exports/` — папка для Excel-файлов, появится сама
- `database.db` — база данных SQLite, появится сама

## Что нужно сделать перед запуском

Откройте `bot.py` и заполните эти строки:

```python
BOT_TOKEN = "PASTE_YOUR_BOT_TOKEN_HERE"
ADMIN_IDS = [123456789]
RESET_PASSWORD = "1234"
CONSULTATION_TEXT = (
    "Для записи на личную консультацию обратитесь по номеру: +7 (900) 000-00-00.\n"
    "Мы поможем подобрать удобное время."
)
```

### Где взять данные
- `BOT_TOKEN` — у `@BotFather`
- `ADMIN_IDS` — ваш Telegram ID
- `RESET_PASSWORD` — любой ваш пароль для сброса базы
- `CONSULTATION_TEXT` — ваш текст для кнопки консультации

## Как установить

```bash
pip install -r requirements.txt
```

## Как запустить

```bash
python bot.py
```

## Как заменить картинку меню
Просто замените файл:

`assets/menu.jpg`

на свою картинку с тем же именем.

## Как загрузить на GitHub
1. Создайте пустой репозиторий на GitHub
2. Загрузите в него все файлы из этой папки
3. Не удаляйте `assets/menu.jpg`, пока не подмените его своей картинкой

## Важно
В этом проекте нет `.env`, поэтому токен лежит прямо в `bot.py`.
Не публикуйте реальный токен в открытом публичном репозитории.

Если репозиторий будет публичным, лучше:
- либо потом перейти на `.env`
- либо сделать репозиторий приватным
